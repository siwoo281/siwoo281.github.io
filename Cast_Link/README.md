# 🏰 Cast-Link - 놀이공원 캐스트 전용 앱

> 마법 같은 하루를 만드는 캐스트들을 위한 디지털 동반자

## ✨ 프로젝트 개요

Cast-Link는 놀이공원 아르바이트 경험에서 영감을 받아 개발한 캐스트 전용 모바일 웹앱 목업입니다. 
실제 근무 중 칭찬메일을 카톡으로 일일이 전달받던 불편함과, 더 나은 캐스트 경험에 대한 아이디어를 구현했습니다.

### 🎯 핵심 기능

- **💌 실시간 칭찬 알림**: 고객 칭찬메일이 바로 앱으로 도착
- **⭐ 서비스스타 시스템**: 칭찬 횟수에 따른 레벨업과 성장 추적
- **💫 샤라웃 (동료 칭찬)**: 캐스트 간 서로 칭찬하고 격려하는 기능
- **👶 미아 알림 시스템**: 분실 아동 발생 시 모든 캐스트에게 즉시 알림
- **📝 근무 관리**: 병가 신청, 교대 요청 등 근무 관련 업무 처리
- **📚 매뉴얼 검색**: 어트랙션별 안전수칙과 현장 꿀팁 아카이브

## 🚀 데모

### 주요 화면

1. **🏰 마법의 홈**: 개인화된 환영 메시지와 오늘의 중요 정보
2. **📅 마법 스케줄**: 하루 근무 일정과 근무 관리 도구
3. **⭐ 나의 성장**: 서비스스타 진행률과 칭찬 메일함
4. **🔔 알림센터**: 미아 발생 등 긴급 알림 관리
5. **📚 마법가이드**: 매뉴얼 검색과 현장 노하우 공유

## 🎨 디자인 특징

### 테마파크 감성
- 🏰 마법, 모험을 연상시키는 카피라이팅
- ✨ 따뜻하고 감성적인 톤앤매너
- 🎠 놀이공원의 즐거움이 묻어나는 인터페이스

### 모바일 우선 설계
- 📱 390×844px 모바일 목업 프레임
- 👆 터치 친화적인 버튼과 인터랙션
- 🎯 한 손으로 쉽게 조작 가능한 하단 네비게이션

## 🛠️ 기술 스택

- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6+)
- **UI Framework**: Font Awesome 6, Pretendard 폰트
- **저장소**: localStorage (클라이언트 사이드 시뮬레이션)
- **알림**: Web Notification API
- **애니메이션**: CSS Transitions & Keyframes

## 🎪 주요 인터랙션

### 샤라웃 보내기
```javascript
// 동료에게 칭찬 메시지 전송
"💫 동료 칭찬하기" 버튼 → 모달 오픈 → 메시지 입력 → 감동 전달!
```

### 미아 알림 시스템
```javascript
// 긴급 상황 시뮬레이션
알림 센터 → "👶 미아 알림 체험" → 인앱 배너 + 브라우저 알림
```

### 서비스스타 레벨업
```javascript
// 칭찬 누적에 따른 성장
칭찬 5개 수집 → 1스타 업그레이드 → "⭐ N-스타 마법사" 달성
```

## 📱 사용 시나리오

### 일반적인 하루
1. 🌅 출근 → 홈에서 오늘 일정 확인
2. 💌 근무 중 → 고객 칭찬 알림 도착 (기분 좋음!)  
3. 💫 휴식 시간 → 도움 준 동료에게 샤라웃 전송
4. 🔔 오후 → 미아 발생 알림, 전체 캐스트 수색 참여
5. 📈 퇴근 → 성장 탭에서 오늘의 칭찬 확인 및 레벨업

### 긴급 상황
1. 👶 미아 발생 → 관리자가 알림 등록
2. 📢 전체 캐스트 앱에 즉시 알림 (인앱 배너 + 푸시)
3. 👥 캐스트들이 협력하여 신속 대응
4. ✅ 발견 완료 → 알림 해제
